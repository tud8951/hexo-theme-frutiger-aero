# Hexo Theme Frutiger Aero

A nostalgic Hexo theme inspired by the Frutiger Aero aesthetic (mid-2000s to early 2010s).

## Features
- **Glossy Glassmorphism**: High-quality glass effects using `backdrop-filter`.
- **Vibrant Gradients**: Bright blues and greens typical of the era.
- **Animated Bubbles**: Floating CSS bubbles for that extra nostalgic feel.
- **Responsive Design**: Works on mobile and desktop.
- **Fast and Lightweight**: No heavy dependencies.

## Installation

1. Go to your Hexo root directory.
2. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/hexo-theme-frutiger-aero themes/frutiger-aero
   ```
3. Update your `_config.yml`:
   ```yaml
   theme: frutiger-aero
   ```

## Configuration
Edit `themes/frutiger-aero/_config.yml` to customize the menu and other settings.
